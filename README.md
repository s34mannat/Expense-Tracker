# 💰 Money Trail

A modern expense tracking web application that helps users manage daily spending, analyze financial habits, and visualize expenses through interactive dashboards.

## 🌐 Live Demo

https://s34mannat.github.io/Expense-Tracker/

---

## 📌 Features

### Expense Management

* Add expenses with name, category, amount, and date
* Delete expenses instantly
* Automatic expense list updates

### Category Management

* Create custom categories
* Delete existing categories
* Dynamic category dropdown updates

### Search & Filtering

* Search expenses by name
* Filter expenses by category
* Real-time results

### Analytics Dashboard

* Total Expenses
* Total Transactions
* Top Category
* Interactive expense visualization using Chart.js

### Data Persistence

* LocalStorage support
* Expenses remain saved after page refresh

### Export Functionality

* Export expense data as CSV

### User Experience

* Responsive design
* Light & Dark Mode
* Modern UI with custom backgrounds
* Smooth navigation between pages
